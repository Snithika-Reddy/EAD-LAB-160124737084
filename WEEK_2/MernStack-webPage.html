Introduction to MERN Stack

MERN stack is a popular web development stack used to build full-stack applications using JavaScript. It consists of MongoDB for database, Express.js for server-side framework, React for building user interfaces, and Node.js for running JavaScript on the server. MERN stack allows developers to use a single language (JavaScript) across the entire application, making development faster, efficient, and easier to maintain.

Technologies involed :

  1. Mongo-DB
  2. Express.js
  3. React.js
  4. Node.js
